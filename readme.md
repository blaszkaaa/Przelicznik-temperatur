# Przelicznik temperatury (Temperature Converter)
---

## Opis po Polsku:

Ten projekt to prosty przelicznik temperatury, napisany w Pythonie. Użytkownik wprowadza wartość temperatury i skalę (Celsius, Fahrenheit lub Kelvin). Klasa `TemperatureConverter` ma metody do konwersji między tymi skalami: `to_celsius`, `to_fahrenheit` i `to_kelvin`. Każda z tych metod przelicza temperaturę na odpowiednią skalę, w zależności od oryginalnej skali. 

## Uruchomienie:

Aby uruchomić ten program, potrzebujesz Pythona 3. 

1. Sklonuj repozytorium
2. Uruchom `python main.py`

## English Description:

This project is a simple temperature converter, written in Python. The user enters the temperature value and the scale (Celsius, Fahrenheit, or Kelvin). The `TemperatureConverter` class has methods for converting between these scales: `to_celsius`, `to_fahrenheit`, and `to_kelvin`. Each of these methods converts the temperature to the appropriate scale, depending on the original scale. 

## Execution:

To run this program, you need Python 3. 

1. Clone the repository
2. Run `python main.py`
